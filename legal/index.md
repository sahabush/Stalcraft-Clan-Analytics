# Пользовательское соглашение сервиса Stalcraft Clan Analytics

**Дата последнего обновления: 13 апреля 2026 года**

## 1. Общие положения

1.1. Настоящее Пользовательское соглашение (далее — «Соглашение») регулирует отношения между Администрацией сервиса Stalcraft Clan Analytics (далее — «Сервис», «Мы») и физическим лицом (далее — «Пользователь», «Вы»), использующим Сервис.

1.2. Сервис представляет собой Discord-бота и сопутствующую инфраструктуру, предназначенную для автоматического анализа статистики клановых войн (CW) в многопользовательской онлайн-игре Stalcraft на основе загружаемых Пользователем скриншотов.

1.3. Начиная использовать Сервис любым способом (авторизация через Discord, загрузка скриншотов, просмотр статистики), Вы подтверждаете, что полностью и безоговорочно принимаете условия настоящего Соглашения. Если Вы не согласны с каким-либо пунктом, Вы обязаны немедленно прекратить использование Сервиса.

1.4. Сервис не является официальным продуктом EXBO, Stalcraft или Discord. Все права на игру Stalcraft и связанные с ней товарные знаки принадлежат их законным правообладателям.

## 2. Авторизация и идентификация

2.1. Для использования Сервиса Пользователь обязан авторизоваться через:
- **Discord OAuth** — для получения уникального идентификатора (`discord_id`), имени пользователя и аватара
- **EXBO OAuth** — для получения доступа к официальному API игры Stalcraft

2.2. Пользователь гарантирует, что использует свои собственные учетные записи Discord и EXBO и не передает доступ к ним третьим лицам.

2.3. Администрация оставляет за собой право отказать в доступе к Сервису любому Пользователю без объяснения причин.

## 3. Сбор, обработка и хранение данных

3.1. **Какие данные мы получаем:**
- Из Discord: `discord_id`, имя пользователя, аватар
- Из EXBO: список персонажей, информацию о клане, ранг Пользователя в клане
- Из загружаемых скриншотов: игровые ники, статистику боев (убийства, смерти, помощь, казна, счет, ранг)

3.2. **Какие данные мы НЕ храним:**
- Пароли от учетных записей Discord и EXBO
- Загруженные скриншоты (удаляются немедленно после успешной обработки)
- Платежные данные (обрабатываются сторонними платежными сервисами)

3.3. **Какие данные мы ХРАНИМ:**
- Статистику клановых войн в агрегированном виде (суммарные и средние показатели)
- Список участников клана, полученный через официальный API Stalcraft
- Метаданные обработанных скриншотов (дата загрузки, уверенность распознавания)

3.4. **Срок хранения данных:** Статистика клановых войн хранится в течение **30 (тридцати) календарных дней**, что соответствует продолжительности одного сезона CW в игре. По истечении этого срока данные автоматически удаляются.

3.5. **Правовое основание обработки:** Обработка данных осуществляется в целях исполнения договора (предоставления доступа к функционалу Сервиса) на основании согласия Пользователя, выраженного путем принятия настоящего Соглашения.

3.6. Пользователь вправе в любой момент запросить удаление всех своих данных, обратившись в службу поддержки через Discord-сервер Сервиса. Запрос будет выполнен в течение 7 рабочих дней.

## 4. Права и обязанности Пользователя

4.1. **Пользователь имеет право:**
- Загружать скриншоты CW для автоматической обработки
- Просматривать агрегированную статистику своего клана
- Редактировать распознанные данные перед сохранением
- Удалять загруженные скриншоты через интерфейс бота
- Запросить полное удаление своих данных

4.2. **Пользователь обязуется:**
- Не загружать скриншоты, содержащие оскорбительный, порнографический, экстремистский или иной незаконный контент
- Не предпринимать попыток взлома, обхода ограничений или нарушения нормальной работы Сервиса
- Не использовать Сервис для нанесения вреда другим Пользователям или третьим лицам
- Не создавать множественные учетные записи для обхода ограничений пробного периода

4.3. Пользователь признает, что загружаемые им скриншоты могут содержать игровые ники третьих лиц (других участников клана), и гарантирует, что имеет законные основания для их обработки в рамках использования Сервиса.

## 5. Условия оплаты

5.1. Сервис предоставляет **бесплатный пробный период** продолжительностью **7 (семь) календарных дней** для каждого клана. Пробный период активируется автоматически при первом выборе клана любым Пользователем.

5.2. По истечении пробного периода дальнейшее использование Сервиса требует оплаты в размере, указанном в момент платежа (на момент публикации — **500 (пятьсот) рублей** за 30 календарных дней).

5.3. Оплата закрепляется за **кланом**, а не за конкретным Пользователем. Любой участник клана, успешно прошедший авторизацию, может пользоваться оплаченным доступом.

5.4. Платеж является **невозвратным**, за исключением случаев, прямо предусмотренных законодательством Российской Федерации, а именно:
- Техническая невозможность использования Сервиса по вине Администрации в течение более чем 48 (сорока восьми) часов подряд
- Двойное (ошибочное) списание денежных средств
- Непредоставление оплаченного доступа после успешной транзакции

5.5. **Не являются основанием для возврата:**
- Изменение состава клана, исключение или добровольный выход Пользователя из клана
- Расформирование (удаление) клана в игре
- Изменения в работе официального API Stalcraft или Discord
- Субъективное недовольство точностью распознавания OCR
- Окончание игрового сезона CW ранее окончания оплаченного периода
- Отсутствие у Пользователя возможности или желания далее пользоваться Сервисом

5.6. Все расчеты производятся в российских рублях через сторонние платежные сервисы. Администрация не получает и не хранит полные платежные данные Пользователя (номер карты, CVV и т.п.).
