# О проекте SCA (SCWA)

> **SCA** (Stalcraft Clan Analytics) и **SCWA** (Stalcraft Clan Wars Analytics) — одно и то же.
> Бот и домен используют SCWA, в документах — SCA.

**SCA (SCWA)** — сервис для автоматического сбора и анализа статистики клановых войн (CW) в игре Stalcraft. Включает в себя **веб-интерфейс** ([scwa.ru](https://scwa.ru)), **Discord-бота**, собственное API, базу данных, модуль оптического распознавания символов (OCR) и **автоматический сканер гранат**.

Проект находится на стадии **активного бета-тестирования**. Текущий функционал непрерывно дорабатывается, внедряются новые возможности, ведётся работа над стабильностью и точностью распознавания.

Это также означает:

- Функционал может изменяться без предварительного уведомления
- Возможны перерывы в работе из-за технических обновлений
- Лимиты на количество запросов могут корректироваться
- Стабильность работы не гарантируется на 100%
- Некоторые функции могут работать нестабильно или временно отключаться

Мы прилагаем все усилия, чтобы минимизировать неудобства и оперативно исправлять ошибки.

---

## Текущие возможности

### Веб-интерфейс ([scwa.ru](https://scwa.ru))
- Личный кабинет с авторизацией через Discord и EXBO
- Загрузка и распознавание скриншотов CW прямо в браузере (с ручной корректировкой перед подтверждением)
- Полная таблица статистики игроков с фильтрами по картам и тирам противника
- Рейтинговая таблица с учётом силы противника
- Карточка игрока: бои, динамика показателей и статистика бросков гранат
- История скриншотов с отметками посещаемости
- Страница сканера гранат: сводка по клану и разбивка по игрокам и этапам

### Discord-бот
- Авторизация, выбор клана, загрузка скриншотов
- Команды статистики: полная таблица, рейтинг, карточка игрока
- Команды по гранатам: статистика за день КВ и сводка за период
- Управление скриншотами для руководства клана (история, экспорт, удаление, массовое отклонение)
- Статус подписки, уведомления и новости сервиса, справка

### Сканер гранат
- Автоматический учёт брошенных гранат по этапам КВ — **без участия игроков**, данные собираются по расписанию
- Сводка по клану за период и детальная разбивка по каждому бойцу
- Доступен и в веб-интерфейсе, и в Discord-боте

### Общее
- Распознавание скриншотов на базе Tesseract OCR (открытый исходный код) с параллельной обработкой
- Экспорт данных в CSV (Excel) — как по боям, так и по гранатам
- Системные уведомления и новости
- Пробный период для ознакомления с сервисом

## Стоимость

На этапе активной разработки стоимость составляет **500 ₽ за 30 дней за клан**.

Цена обоснована необходимостью финансирования:
- Аренда серверной инфраструктуры;
- Обеспечение вычислительных ресурсов для OCR-обработки и работы сканера;
- Разработка и поддержка функционала.

Для ознакомления доступен **пробный период**. По мере выхода проекта на плато — когда основной функционал будет реализован и отлажен — стоимость **возможно будет снижена**. Финальная цена определяется из расчёта поддержания инфраструктуры и минимальной маржи, достаточной для дальнейшего развития.

> Одна подписка предоставляет доступ всем участникам клана. Рядовым игрокам оплата не требуется.

Поддержать проект также можно добровольным донатом — это помогает развитию и удержанию цены.

## Планы развития

**Недавно реализовано:**
- Веб-интерфейс для доступа к статистике без Discord
- Сканер гранат с автоматическим сбором данных
- Экспорт статистики гранат в CSV

**Ближайшие задачи:**
- Улучшение алгоритмов OCR на основе накопленных данных
- Расширенная аналитика по отрядам внутри клана
- Дальнейшее развитие веб-интерфейса и его удобства

**Среднесрочные планы:**
- Мобильная версия веб-интерфейса
- Расширение аналитики и визуализаций

**Долгосрочные планы:**
- Публичное API для интеграции со сторонними сервисами
- Поддержка нескольких регионов (RU, EU, NA)

---

## Позиция проекта

**SCA (SCWA) — инструмент сбора данных, а не оценка боевых качеств.** Статистика не учитывает роль игрока в бою, тактические задачи и командное взаимодействие. Сервис помогает собирать информацию и выявлять аномалии, но не заменяет опыт и интуицию руководства клана.

Проект **не поддерживает** использование статистики для травли или публичного порицания игроков.

## Безопасность

Сервис работает исключительно с легальными источниками данных: официальным API EXBO и добровольно загружаемыми скриншотами. Мы не используем запрещённое программное обеспечение и не нарушаем пользовательское соглашение Stalcraft.

Подробнее — в [Политике конфиденциальности](https://docs.scwa.ru/privacy).
