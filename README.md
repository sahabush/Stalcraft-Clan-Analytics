**SCWA** — сервис аналитики клановых войн (КВ) в STALCRAFT / STALZONE: Discord-бот
и веб-интерфейс. Распознаёт скриншоты таблиц боёв (OCR), ведёт статистику игроков и
кланов, строит графики, помогает планировать КВ.
