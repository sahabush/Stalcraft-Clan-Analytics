## 2. Термины и определения

2.1. **Сервис** — программный комплекс, состоящий из Discord-бота, API-сервера, базы данных и OCR-воркера, обеспечивающий функционал анализа статистики.

2.2. **Пользователь** — физическое лицо, достигшее возраста 14 лет, использующее Сервис.

2.3. **Клан** — игровое объединение в игре Stalcraft, статистика которого анализируется Сервисом.

2.4. **Скриншот** — изображение, загружаемое Пользователем, содержащее таблицу результатов клановой войны.

2.5. **Лидер/Полковник** — Пользователь, имеющий соответствующий ранг в клане согласно API Stalcraft, обладающий расширенными правами в Сервисе.

## 3. Авторизация и идентификация

3.1. Для использования Сервиса Пользователь обязан авторизоваться через:
- **Discord OAuth** — для получения уникального идентификатора (`discord_id`), имени пользователя и аватара;
- **EXBO OAuth** — для получения доступа к официальному API игры Stalcraft.

3.2. Пользователь гарантирует, что использует свои собственные учетные записи Discord и EXBO и не передает доступ к ним третьим лицам.

3.3. Администрация оставляет за собой право отказать в доступе к Сервису любому Пользователю без объяснения причин, в том числе путём блокировки (бана) через административную панель.

3.4. При блокировке Пользователя ему запрещается доступ ко всем функциям Сервиса. Информация о блокировке сохраняется в базе данных Сервиса.

## 4. Сбор, обработка и хранение данных

4.1. **Какие данные мы получаем:**
- Из Discord: `discord_id`, имя пользователя, аватар;
- Из EXBO: список персонажей, информацию о клане, ранг Пользователя в клане;
- Из загружаемых скриншотов: игровые ники, статистику боёв (убийства, смерти, помощь, казна, счёт, ранг).

4.2. **Какие данные мы НЕ храним:**
- Пароли от учетных записей Discord и EXBO;
- Загруженные скриншоты (удаляются немедленно после успешной обработки либо по команде Пользователя);
- Платёжные данные (обрабатываются сторонними платёжными сервисами);
- Личную переписку Пользователей.

4.3. **Какие данные мы ХРАНИМ:**
- Статистику клановых войн в агрегированном виде (суммарные и средние показатели);
- Список участников клана, полученный через официальный API Stalcraft;
- Метаданные обработанных скриншотов (дата загрузки, уверенность распознавания);
- Информацию о блокировках Пользователей (для предотвращения обхода бана).

4.4. **Срок хранения данных:** Статистика клановых войн хранится в течение **30 (тридцати) календарных дней**, что соответствует продолжительности одного сезона CW в игре. По истечении этого срока данные автоматически удаляются.

4.5. **Правовое основание обработки:** Обработка данных осуществляется в целях исполнения договора (предоставления доступа к функционалу Сервиса) на основании согласия Пользователя, выраженного путём принятия настоящего Соглашения.

4.6. Пользователь вправе в любой момент запросить удаление всех своих данных, обратившись в службу поддержки через Discord-сервер Сервиса. Запрос будет выполнен в течение 7 (семи) рабочих дней.

5.3. Пользователь признает, что загружаемые им скриншоты могут содержать игровые ники третьих лиц (других участников клана), и гарантирует, что имеет законные основания для их обработки в рамках использования Сервиса.

5.4. **Лимиты использования:**
Сервис устанавливает следующие ограничения для предотвращения перегрузок:
- Загрузка скриншотов: не более 10 за 5 минут;
- Просмотр статистики: не более 10 запросов в минуту;
- Детальная статистика игрока: не более 15 запросов в минуту;
- Экспорт данных: не более 5 запросов в минуту.

При превышении лимитов Пользователь получает уведомление с указанием времени ожидания.
