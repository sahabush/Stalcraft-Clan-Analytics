## 2. Термины и определения

2.1. **Сервис (SCA/SCWA)** — программный комплекс, доступный по адресу [bot.scwa.ru](https://bot.scwa.ru), включающий Discord-бота, API-сервер, базу данных и OCR-воркер.

2.2. **Пользователь** — физическое лицо, достигшее 14 лет, использующее Сервис.

2.3. **Клан** — игровое объединение в Stalcraft, статистика которого анализируется Сервисом.

2.4. **Скриншот** — изображение таблицы результатов CW, загружаемое Пользователем.

2.5. **Лидер/Полковник** — Пользователь с соответствующим рангом в клане согласно API Stalcraft, обладающий расширенными правами.

---

## 3. Авторизация и идентификация

3.1. Для использования Сервиса необходима авторизация через:
- **Discord OAuth** — получение `discord_id`, имени пользователя и аватара;
- **EXBO OAuth** — получение доступа к официальному API Stalcraft.

3.2. Пользователь гарантирует, что использует собственные учётные записи и не передаёт доступ к ним третьим лицам.

3.3. Администрация оставляет за собой право отказать в доступе любому Пользователю без объяснения причин.

3.4. При блокировке Пользователю запрещается доступ ко всем функциям Сервиса.

---

## 4. Сбор, обработка и хранение данных

4.1. **Какие данные мы получаем:**
- Из Discord: `discord_id`, имя пользователя, аватар;
- Из EXBO: список персонажей, информация о клане, ранг;
- Из скриншотов: игровые ники, статистика боёв.

4.2. **Какие данные мы НЕ храним:**
- Пароли от Discord и EXBO;
- Загруженные скриншоты (удаляются после обработки);
- Полные платёжные данные (обрабатываются сторонними сервисами);
- Личную переписку Пользователей.

4.3. **Какие данные мы ХРАНИМ:**
- Агрегированную статистику CW;
- Список участников клана (через API Stalcraft);
- Метаданные скриншотов (дата, уверенность распознавания);
- Информацию о блокировках.

4.4. **Срок хранения:** статистика CW — 30 дней. По истечении данные автоматически удаляются.

4.5. Пользователь вправе запросить удаление всех своих данных через службу поддержки. Срок выполнения — до 7 рабочих дней.

4.6. Подробнее — в [Политике конфиденциальности](https://docs.scwa.ru/privacy).

4.7. Пользователь уведомлён и согласен с тем, что обработка данных осуществляется на серверах, расположенных на территории Федеративной Республики Германия. Состав передаваемых данных ограничен информацией, необходимой для функционирования Сервиса.

## 5. Права и обязанности Пользователя

5.1. **Пользователь имеет право:**
- Загружать скриншоты для обработки;
- Просматривать статистику клана;
- Редактировать распознанные данные;
- Экспортировать статистику в CSV;
- Просматривать детальную статистику игроков;
- Запросить удаление своих данных.

5.2. **Пользователь обязуется:**
- Не загружать скриншоты с незаконным содержанием;
- Не пытаться взломать или обойти ограничения Сервиса;
- Не использовать Сервис для нанесения вреда другим;
- Соблюдать установленные лимиты (rate limits).

5.3. Подробнее о лимитах — в [Руководстве пользователя](https://docs.scwa.ru/GUIDE).

## 6. Права и обязанности Лидеров/Полковников

6.1. Пользователи с рангом **LEADER** или **COLONEL** получают дополнительные права:
- Просмотр списка всех скриншотов клана;
- Массовое отклонение неподтверждённых скриншотов;
- Удаление дубликатов и полная очистка статистики;
- Экспорт данных любого скриншота;
- Просмотр истории с поскриншотной навигацией и посещаемостью.

6.2. Лидеры/Полковники обязуются использовать эти права в интересах клана.

7.4. **Возврат средств** возможен в случаях:

- Непредоставление оплаченного доступа.
